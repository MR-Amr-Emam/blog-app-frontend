# Blog App Frontend

### generally consisted of four pages

1- **Home Page**
2- **Profile Page**
3- **Group Page**
4- **chat Page**

to run just wrtie in the command line in the root directory
```
npm run start
```

to make changes you can run dev server temporary to debug and rebuild the app
```
npm run build
```
